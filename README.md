作者寄语:
此软件开源且免费纯净，用Visual Studio Code编写的，现在是第一版，我首先实现的是用现代的webview2在新窗口加载元梦之星云游戏；
因为无论是Chrome还是Edge浏览器，还是安卓模拟器，资源占用都高；
于是选用webview2的形式加载网页，最大程度降低资源占用，实现多开，且.Net 8.0对Webview2支持度较好；

目前只单纯的实现低资源加载腾讯先锋网页云游戏，此后还要逐步加入自动化脚本，录制操作等功能。
