日期: 2025年2月7日 15:49
版本号: 1.0.0
开发者：星宝@Sakura

作者寄语:
此软件开源且免费纯净，用Visual Studio Code编写的，现在是第一版，我首先实现的是用现代的webview2在新窗口加载元梦之星云游戏；
因为无论是Chrome还是Edge浏览器，还是安卓模拟器，资源占用都高；
于是选用webview2的形式加载网页，最大程度降低资源占用，实现多开，且.Net 8.0对Webview2支持度较好；

目前只单纯的实现低资源加载腾讯先锋网页云游戏，此后还要逐步加入自动化脚本，录制操作等功能。
元梦之星云游戏官网 https://gamer.qq.com/v2/game/96897?ichannel=pcgames0Fpcgames1

日志记录
日期: 2025年2月10日 19:49
版本号: 1.1.0
开发者：星宝@Sakura
此版本更新了很多东西

- 用推荐的容器，把窗口分为了左侧网页区，右侧控制区
- 限定窗口大小为1500*720，其中左侧网页区1280x720，右侧控制区220x720
- 底部新增了Github、Youtube、Bilibili三个图标，点击即可跳转到对应的网址
- 控制区置顶文本框为预设文案，且不可用户编辑，是作者的寄语
- 控制区三个按钮分别为：刷新、一键挂机/停止挂机、关于作者
- 当点击刷新后，调用 WebView2 的 Reload 方法刷新腾讯先锋网页云游戏
- 一键挂机内置了简单的挂机操作，并支持后台执行，不受电脑锁屏影响，
- 关于作者点击开来，也会有作者的寄语和赞赏码
